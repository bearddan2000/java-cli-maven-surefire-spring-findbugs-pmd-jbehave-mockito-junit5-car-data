# java-cli-maven-surefire-spring-findbugs-pmd-jbehave-mockito-junit5-car-data

## Description
A POC for maven app using junit5
and jbehave and mockio framework
 with findbugs,
pmd and urefire plugin.

## Tech stack
- java
- maven
	- spring
	- mockito
  - junit5
  - findbugs
  - pmd
  - surefire
  - jbehave

## Docker stack
- adoptopenjdk/maven-openjdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
